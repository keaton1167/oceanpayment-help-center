## Homepage Guide

### 1. Home Page Interface Description

- Click on the 'OPCOUNT' section on the left to access the homepage interface on the ODPM.  
Interface Description: OPCCOUNT is divided into 【Acquiring Account】 & 【Global Accounts】. If your company has activated the corresponding accounts, the respective account sections will be displayed. For details, please contact your dedicated Account Manager or Business Manager.

- 【Acquiring Account】Homepage displays the currency and amount, along with real-time transaction records for account verification, withdrawals, and POBO.
![图片](/img/help-center/guideline-of-opccount-new-oldn/KPLlbZQiGom79dxmTB1caGEWnoc.png)

- 【Global Account】Homepage (if applicable) displays billing currency amounts and real-time operation records for account review, withdrawals, POBO. It also features currency exchange rate query and quick access.
![图片](/img/help-center/guideline-of-opccount-new-oldn/C96eb0Zogoxi2sxOkWyc9CVWnfg.png)

- Funds will first be settled into the 【Acquiring Account】. If you require funds to be synchronized to your 【Global Account】 (if applicable), you must first enable “Sync To Global Account” in the 【Acquiring Account】. Activate the “Should the funds in the acquiring account be automatically withdrawn to the global account?” button, configure the “Automatic Withdrawal Time” and “Select Withdrawal Currency”. Note that once enabled, funds will be withdrawn to the 【Global Account】 at 12:00 PM daily. Only proceed with withdrawal after funds have been synchronized to the 【Global Account】.
![图片](/img/help-center/guideline-of-opccount-new-oldn/DdoIbznDYoh74oxYifPc6Y7Wnkb.png)

- 【Global Account】(if applicable) Exchange rate query function description: View current reference exchange rates based on account currency and withdrawal currency.
- 【Global Account】(if applicable) Quick access function description：
1. Withdrawal Application: Can initiate a withdraw application
2. POBO Application: Can initiate a POBO application 
3. Collection Report: Can check the OPPCOUNT collection report 
4. Automatic Withdrawal: Can configure the automatic withdrawal function 
5. Withdrawal Account: Can manage the withdrawal account 
6. POBO Account: Can manage the POBO account 

- Amount description:
（1）Available amount: Amount that can be initiated for withdrawal

（2）Frozen amount: The amount of money that needs to be temporarily frozen in the account of OPCCOUNT platform due to compliance and other reasons, and cannot be initiated to withdraw

### 6. Target Exchange Rate Withdrawal Setting

### 7. This feature primarily allows you to set your target exchange rate for withdrawals within the valid timeframe. The system will automatically monitor exchange rates in real time. If the real-time rate matches your target rate, the system will automatically withdraw funds to your bank account at the target rate.

### 8. Set Target Exchange Rate Withdrawal: Click【Target Exchange Rate Withdrawal】and select the corresponding account type.

![图片](/img/help-center/guideline-of-opccount-new-oldn/AHVAbmnSOoqYfux9yZ0c51vZngf.png)

- The bank account information displayed by default includes the receiving bank, account holder name, account number, and withdrawal currency upon successful withdrawal.
The available withdrawal currencies are those currently supported for withdrawal from the merchant account (i.e., the original currency before conversion).

Exchange relationship: The withdrawal currency is converted from the available withdrawal currency to the withdrawal account's designated currency.

![图片](/img/help-center/guideline-of-opccount-new-oldn/AD8ab81Hfo7Id6xmGz7cmoF1nqC.png)

- You may select other bank accounts on this page.
![图片](/img/help-center/guideline-of-opccount-new-oldn/NR7Xb8898oY5bPxBYK6cGpXKnfe.png)

- On this interface, enter the cryptocurrency to withdraw, target exchange rate, target withdrawal amount, and effective deadline, then click Submit.

- A confirmation prompt will appear upon successful submission.

- Simultaneously, the OPCCOUNT withdrawal report will add a new record for this withdrawal with the type “Target Withdrawal” and status “Pending.” If the target exchange rate is not reached within a reasonable timeframe or you wish to modify the rate, click “Cancel Task” to terminate this withdrawal and set a new exchange rate.
![图片](/img/help-center/guideline-of-opccount-new-oldn/AOLxbdzjWoYLvixoyB0c8OJanNh.png)

- Important Notes: When the target exchange rate is not achieved, the following fields will display null values: batch number, withdrawal time, exchange rate, handling fee, and paying bank. Only one exchange rate task per currency is supported for the same withdrawal account at any given time.

- Example: If an account's withdrawal currency is USD and HKD is available for withdrawal, only one HKD-to-USD withdrawal task may be submitted for that account. 


- When the system detects that the exchange rate has fluctuated to the target rate, the record will be updated to the status: Pending. Simultaneously, it will generate the withdrawal batch number, withdrawal time, exchange rate (actual withdrawal rate), handling fee, and other relevant information.
![图片](/img/help-center/guideline-of-opccount-new-oldn/DMkNbJantomRvwx7mpFc5grlnkc.png)

## Splitting Guide

### 18. Introduction

- Splitting is mainly used to handle fund allocation through terminal numbers.
### 19. Function Description

- Click 【Splitting】 - 【Bill Splitting】 to access the Splitting function

![图片](/img/help-center/guideline-of-opccount-new-oldn/VEZrb9FZIoTSzexqbu8c42uEnth.png)

- Page allows filtering and includes buttons for【New】,【Upload】,【Download】,【Submit for Approval】, and【Delete】. You can also modify or download documents for existing Splitting records.

![图片](/img/help-center/guideline-of-opccount-new-oldn/JkVNb3xo3owviRxY7vCcP8pzn2z.png)

Button Description:

New: Click New to initiate a Splitting application.

Upload: Click Upload to download the corresponding template for batch uploading Splitting applications.

Download: Select Splitting application records on the current page to download Splitting details in bulk.

Submit for Approval: Select newly added Splitting applications on the current page to submit to Oceanpayment for approval.

Delete: Delete the selected Splitting application.

Download Document: Download the corresponding Splitting document.

### 20. Splitting Process

- Click【New】to go to the information entry page. Fill in the corresponding Terminal No, beneficiary account, and upload the related invoice/contract, then click Submit.

Notes:

Terminal No: The Terminal No. currently integrated with your website.

Beneficiary's Split Ratio: The allocation percentage of settlement for the beneficiary under this terminal.

Transaction Fees Bearing Method: BEN (borne by the recipient), OUR (borne by the payer), SHA (shared).

Limit Control: The maximum amount that can be allocated to the beneficiary per transaction.

Settlement Cycle: The time period for processing the split. For example, if the cycle is 3 days, a payout with splitting will be processed after 3 days with the corresponding fees deducted.

Beneficiary’s bank account must be whitelisted in Oceanpayment. To initiate a split, please contact your account manager.

![图片](/img/help-center/guideline-of-opccount-new-oldn/UWEbbEIVAoVw0vxEEwycX8oBn6c.png)
