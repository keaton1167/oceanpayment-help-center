Mastercard plans to introduce an updated version of chargeback reasons starting from April 11, 2025. The updated version falls into the following major categories:

- Authorization-Related Chargeback
- Cardholder Disputes
- Duplicate Processing
- Chip Liability Shift-Lost / Stolen / Never Received Issue ( NRI ) Fraud
- Chip Liability Shift
- Questionable Merchant Activity
- No Cardholder Authorization

Please check the details in [Update on Mastercard Chargeback Reasons](/files/help-center/wordpress-attachments-en/attachment-11.pdf)
