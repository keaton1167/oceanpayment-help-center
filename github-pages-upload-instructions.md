# GitHub Pages upload instructions

Upload or commit these files to the `main` branch of:

`https://github.com/keaton1167/oceanpayment-help-center`

## Files to update

- `.github/workflows/deploy-github-pages.yml`
- `.gitignore`
- `docusaurus.config.js`
- `src/components/DocsHome/index.js`

## GitHub Pages setting

In repository settings:

1. Open `Settings`.
2. Open `Pages`.
3. Set `Source` to `GitHub Actions`.

## After upload

1. Open the `Actions` tab.
2. Select `Deploy GitHub Pages`.
3. Run the workflow or wait for the push trigger.
4. After it succeeds, open:

`https://keaton1167.github.io/oceanpayment-help-center/`
