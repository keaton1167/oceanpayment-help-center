## 批量申诉功能介绍

### 1. 功能介绍

批量申诉功能支持商户自主定义申诉资料的排版及内容，并通过批量操作减少多步骤操作，提升申诉资料提交效率。

### 2. 批量申诉与发起申诉的区别

- **发起申诉**：对拒付订单单笔提交申诉资料。在拒付订单下点击“发起申诉”后，进入指定的申诉提交页面，根据系统指定的文本框进行编辑并上传证明材料。
- **批量申诉**：拒付交易板块的批量功能，支持为多个支付 ID 提交申诉资料，同时减少系统排版限制，允许商户自定义编辑申诉文档。

批量申诉和发起申诉均属于提交申诉资料的方式，最终申诉结果仍取决于银行。

### 3. 操作路径

进入 `ODPM > Payment > 异常交易管理 > 拒付交易`。在拒付交易板块的当前页面，可见“批量申诉”按钮。

![批量申诉操作入口](/img/help-center/batch-chargeback-representment-guide/image6.png)

### 4. 批量申诉说明

为保证申诉文档准确匹配到需要申诉的拒付订单，请根据以下要求整理和提交材料：

- 上传文件前，请先将文件压缩为 ZIP 格式。
- 压缩文件命名格式必须为 `Representment-YYYYMMDD.zip`。
- 压缩包内的申诉文档需以支付 ID 命名，支持 1000 个以内。
- 不支持重复支付 ID 的申诉文件提交。如同一支付 ID 存在两笔拒付需要申诉，仅支持单笔提交。
- 申诉文档格式要求请参考示例文档。
- 提交后，对应支付 ID 的拒付状态变更为“申诉中”，即代表文件上传成功。
- 上传成功后，可点击“下载凭证”预览申诉文档。

### 5. 示例文档

点击“批量申诉”，可在弹窗页面获取上传示例文档，并参照示例内容整理申诉证明材料。

![批量申诉弹窗与示例文档入口](/img/help-center/batch-chargeback-representment-guide/image7.png)

![批量申诉示例文档](/img/help-center/batch-chargeback-representment-guide/image8.png)

#### 5.1 申诉格式

- 使用文字描述并配合相应图片佐证。
- 文字必须使用英文。
- 字体要求：10 号微软雅黑。

#### 5.2 申诉图片

- 申诉资料的图片必须清晰且大小合适。
- 图片支持 JPG、JPEG、PNG 格式。
- 单个图片最大 3 MB。
- 图片尺寸不超过 1800 px × 1500 px。

#### 5.3 申诉文档限制

- Word 文件总页数不超过 20 页。
- 文件大小不得超过 15 MB。
- 最终提交的 Word 申诉文档需要以支付 ID 命名。
- 模板中必填部分的文字描述仅供参考，选填部分可根据实际情况补充调整。
- 提交材料前，请删除示例模板中的中文及补充高亮字体内容。

### 6. 如何批量申诉

1. 点击“批量申诉”。
2. 在弹窗页面选择准备好的压缩文件。
3. 点击“提交”。
4. 提交成功后，可点击“下载”查阅。

![上传并提交批量申诉文件](/img/help-center/batch-chargeback-representment-guide/image10.png)

![批量申诉下载凭证入口](/img/help-center/batch-chargeback-representment-guide/image11.png)

### 7. 上传按钮限制规则

- **压缩文件命名**：`Representment-YYYYMMDD.zip`，例如 `Representment-20250820.zip`。
- **压缩包数量上限**：1 个。
- **压缩包内文件数量上限**：1000 个。
- **压缩包内单个文件大小**：10 MB。
- 压缩包内不允许上传重复支付 ID 的申诉文件。
- 同一支付 ID 存在多笔拒付时，不支持批量申诉，仅支持单笔发起申诉。
- 支付 ID 需满足拒付状态为“待处理”。

### 8. 常见问题（FAQ）

**Q：我可以一次申诉多少笔交易？**

A：单次上传的 ZIP 压缩包内最多支持 1000 个文件，即 1000 个支付 ID。

**Q：上传文件后，如何确认提交成功？**

A：找到刚提交申诉的支付 ID，检查其“拒付状态”。若状态由“待处理”变更为“申诉中”，则代表文件已成功上传并被系统接收。

**Q：如果误传文件或需要补充材料怎么办？**

A：系统不支持通过批量申诉重复提交相同支付 ID。如操作失误或需要补充材料，可通过对应支付 ID 的“发起申诉”按钮重新单笔提交。提交成功后，新内容将覆盖上一次提交的申诉内容，并以最后一次更新时间为准。

**Q：点击“下载凭证”后看到的内容是什么？**

A：系统会以 PDF 格式展示本次提交生成的确认回执，供记录和核对。

**Q：可以上传 PDF、JPG、TIFF 等其他格式的申诉文档吗？**

A：不支持，目前申诉文档仅支持 `.docx` 格式。
