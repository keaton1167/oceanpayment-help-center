**一、功能入口：IAM > 首页 > OPCCOUNT业务申请**

![图片](/img/help-center/iam申请opccount全球账户业务操作手册-2026年4月20日-7jdu/Pj90bZhCioTxoLxmTLdcxPFJnQg.png)

- 功能说明：商户进行企业认证完成后，使用邮箱账号登录进入IAM系统，点击“OPCCOUNT业务申请”，可进行业务申请。
- IAM登陆网址：[https://iam.oceanpayment.com/login](https://iam.oceanpayment.com/login)

**二、OPCCOUNT业务申请**

- 操作说明：点击“OPCCOUNT业务申请”入口后，进入相关页面，填写业务用途、并上传相关资料，完成后点击提交申请。
![图片](/img/help-center/iam申请opccount全球账户业务操作手册-2026年4月20日-7jdu/OG6kboCRBoO5FAxsxgvc3oIenyr.png)

![图片](/img/help-center/iam申请opccount全球账户业务操作手册-2026年4月20日-7jdu/DpU9bRW8BoZnTTxZKGNcqWCwnBd.png)

**三、审核结果通知**

- 提交申请后，若审核不通过，会在【IAM > 首页】进行相关提示。此时点击“立即申请”、“去处理”按钮，可重新进入业务申请页面，重新进行资料填写。
![图片](/img/help-center/iam申请opccount全球账户业务操作手册-2026年4月20日-7jdu/JE1BbmvANoHHfYx2PRccH7binqd.png)

- 若审核通过，且开通对应账号后，可登陆IAM系统，通过链接进入OPCCOUNT系统进行提现/代付操作，需注意，开通OPCCOUNT全球账户业务，默认同步开通外贸收款业务。
![图片](/img/help-center/iam申请opccount全球账户业务操作手册-2026年4月20日-7jdu/YcW7bRhDwojexbxfZk3cnvbknoe.png)
