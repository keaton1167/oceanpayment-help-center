## 1. 账户后台发起网站申请后，后续流程是什么：

当贵司在ODPM账户后台发起网站申请后，专属客户关系经理会发起对应网站接入，会在一个工作日左右给出对应网站网修，网修完成后，提交银行通道申请。

## 2. 什么情况下需要发起网站申请：

- 未在Oceanpayment申请接入过的网站，
- 网站在Oceanpayment申请接入过，但后续域名发生了变更，需要重新申请，因为通道配置信息对应的是网站域名，因此如果域名发生了变更，则需要重新申请新的支付通道。

## 3. 账户后台发起网站申请操作流程： 

<ol type="a">
<li>登陆ODPM账户后台，登陆网址：https://odpm.oceanpayment.com</li>
</ol>

- 登陆ODPM账户后台后，点击【Payment】-【网站管理】-【新增】
![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/HwRHb8537odeBgxk0e3cs6AhnUc.png)

<ol start="2" type="a">
<li>填写申请网站信息</li>
</ol>

- 按页面内容填写对应申请网站的基础信息及网站信息，可同时新增多个网站，
基础信息：填写申请网站的推广国家、交易币种、网站客单价

网站信息：填写申请网站的域名、产品类型、建站模式、备注及域名证书，其中域名格式为XXX.com，选择建站模式后会提示上传域名证书，可上传域名证书或者店铺后台域名截图。

![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/Ywq0byq9loL8Zxx4rTmcYsqJnzc.png)

填写完出现【操作成功】代表该网站提交成功，如提示【操作失败】，请确认该站点是否重复提交或者联系您的专属客关咨询。

![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/YKgVbePhdoowvtxtsDwcFUCHnrg.png)

<ol start="3" type="a">
<li>下载网修处理及网修完成</li>
</ol>

- 当网站申请提交后，贵司需要在两个工作日左右在账户后台-网站管理找到对应网站申请记录，点击下载按钮，根据下载的网修文档针对网站进行对应修改，修改过程中如有任何疑问，可随时联系您的专属客关咨询，
- 网修完成中，点击网修完成即可，
- 当需要取消该站点接入时，点击取消接入即可。
![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/PqrKbQ991o9qvqx6vy2cJwZMnWf.png)

- 当遇到批量新增网站申请时，有多个网修的情况，可在【网站管理】页面点击下载网修文件，可批量下载对应已出网修。
![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/Hrg9bGlW4om27rxMJ9IcmUbUnIb.png)

- 当需要下载已申请网站的数据时，可点击下载按钮。
![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/RhSub7EvIonWI6xQuD2cW6BbnYf.png)

<ol start="4" type="a">
<li>查看网站申请进度</li>
</ol>

- 可点击高级筛选进行自定义查询筛选，
- 网站审核状态：网站提交后，我司内部会进行初步审核。
- 网修说明：初审之后，会根据网站内容给出对应的网修方案，贵司需进行下载网修进行
网站修改，完成修改之后点击“完成网修”即可。
- 通道申请状态：代表我司内部复核通过，正在银行阶段审核。
- 转正状态：银行审核通道已下达，我司内部进行通道验收及上线流程。
- 终端号状态：状态为“正常”即代表该任务已完成，通道已上线；状态为“未激活/测试”代表任务还在进行中；状态为“停用”代表该通道已关停处理，无法继续使用。
![图片](/img/help-center/oceanpayment-odpm-账户后台网站申请指引-zlk1/Bh69boie9ov8MgxVY5pcG2EgnDc.png)
