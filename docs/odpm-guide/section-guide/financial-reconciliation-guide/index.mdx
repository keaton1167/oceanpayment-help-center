## 财务专区

### 1. 账户资金动态

掌握账户整体资金情况，如图所示：

![账户资金动态页面](/img/help-center/financial-reconciliation-guide/image2.png)

显示数据为结算币种与记账金额：

1. **累计划款金额**：实际划款金额累积合计（已扣费）。
2. **累计未结算金额**：还未划款的订单金额（未扣费用，包含可结算金额）。
3. **累计未结算的保证金**：已结算订单留存的循环保证金、固定保证金与押金。
4. **可结算金额**：已满足结算周期且当前符合对应结算方式的金额。

- 商户结算方式为“审单结算 + 上传单号”时，展示金额为满结算周期内，已上传物流单号且物流审核已妥投的订单金额。此金额仅供参考，会随物流状态变更而更新。
- 商户结算方式为“免审单 + 上传单号”时，展示金额为满结算周期内且已上传物流单号的订单金额。此金额仅供参考，会随商户上传物流单号而更新。

### 2. 交易明细查询

可按不同条件及时间段查询交易订单。系统按交易类型以正负数明细显示，方便查看统计数据，并支持下载交易明细 Excel/CSV 表。查询筛选条件如下：

![交易明细查询条件](/img/help-center/financial-reconciliation-guide/image3.png)

1. **支付方式**：Credit Card、其他已开通的支付方式。
2. **交易类型**：成功、退款、拒付、调单、调单取消、申诉成功、失败。
3. **时间类型**：记账时间，即结算及报表统计的时间。
4. **物流单号**：可筛选查询已同步或尚未同步物流信息的订单。
5. **妥投状态**：
   - 空：此订单尚未审核物流状态或尚未上传物流信息。
   - 未妥投：此订单同步的物流单号状态为包裹尚未成功签收。
   - 已妥投：此订单同步的物流单号状态为包裹已签收。
   - 异常：此订单同步的物流单号状态异常，需核查是否上传同步错误，可重新上传更新。
6. **划款状态**：已划款、未划款。可通过此条件筛选统计已结算、未结算的订单金额情况。
7. **记账金额**：交易金额按交易当天的汇率转换成结算币种的金额，结算及计费以此金额为基础。
8. **金额统计**：按所选时间、状态等条件查询的订单合计金额（记账金额与交易金额）。
9. **SaaS Fee**：按所选时间、状态等条件下载订单统计使用的 SaaS Fee（平台）费用合计金额。

> 注：物流单号与妥投状态的查询条件仅针对“成功”交易类型的订单生效。

### 3. 交易划款查询

可查询每个划款批次的情况，并下载账单及明细。

![交易划款查询页面](/img/help-center/financial-reconciliation-guide/image4.png)

可通过划款币种、划款类型、划款时间查询划款批次情况：

- **下载对账单（PDF）**：下载 PDF 对账单，核对本批次划款情况。此功能仅适用于交易划款类型，对账单样式见“报表账单样式说明”。
- **下载小结（Excel）**：下载对应批次的汇总表。如需批量下载，请勾选所需批次，点击左上方“下载小结”。
- **下载明细（Excel/CSV）**：下载对应批次每一笔订单的扣费及划款明细。如需批量下载，请勾选所需批次，点击左上方“下载明细”。

点击对应的“划款批次号”，可展开隐藏信息，查看本次划款的付款银行及额外备注信息。

![划款批次详情](/img/help-center/financial-reconciliation-guide/image5.png)

## 报表管理

### 1. 交易报表

每月 1 日可下载上个月的交易报表。报表以 Excel 格式统计上月交易金额数据，并按支付方式、交易类型、交易币种、记账币种分类。

![交易报表下载页面](/img/help-center/financial-reconciliation-guide/image6.png)

### 2. 结算报表

每月 1 日可下载上个月的结算月报，方便财务统计数据及做账。

![结算报表下载页面](/img/help-center/financial-reconciliation-guide/image7.png)

按所需月份下载结算报表。

## 报表账单样式说明

### 1. 对账单（Statement）

![对账单样式](/img/help-center/financial-reconciliation-guide/image8.png)

- **已结算交易款**：本批次已结算的交易，包含不同交易类型，账单中会分别显示。
- **费用**：本批次结算订单扣取的费用，在划款明细表中对应每笔订单均有体现。
- **内扣费用**：例如月费、年服务费等，具体项目会在划款备注中说明。
- **其他款项**：例如银行转账费、押金及其他调账金额，具体项目会在划款备注中说明。
- **保证金**：本批次结算所扣留的保证金和押金。
- **实划金额**：已结算交易款 + 费用 + 其他款项 + 保证金。

### 2. 交易报表

![交易报表样式](/img/help-center/financial-reconciliation-guide/image9.png)

- **统计时间段**：记账时间为当月 1 日至 31 日的所有交易数据。
- **汇总金额**：与结算月报中的“本月新增交易款”一致。
- 可按支付方式、交易类型、交易币种、记账币种进行不同维度的统计。
- **拒付率（参考）**：
  - 拒付率（笔数）= 本月发生的拒付笔数 / 本月成功交易笔数。
  - 拒付率（金额）= 本月发生的拒付金额 / 本月成功交易金额。金额按次月 1 日当天的汇率换算，此数据仅供参考。
- **退款率（参考）**：
  - 退款率（笔数）= 本月处理的退款笔数 / 本月成功交易笔数。
  - 退款率（金额）= 本月处理的退款金额 / 本月成功交易金额。金额按次月 1 日当天的汇率换算，此数据仅供参考。

### 3. 结算月报

![结算月报样式](/img/help-center/financial-reconciliation-guide/image10.png)

- 按不同结算币种分页显示。
- 当月只能下载上个月及更早月份的报表数据。
- **交易款/保证金余额**：截至本月底剩余未结算的交易款/保证金金额。
  - 交易款期初余额 = 上个月交易款期末金额。
  - 交易款期末余额 = 期初余额 + 本月新增交易款 - 本月已结算交易款。
  - 保证金期初余额 = 上个月保证金期末金额。
  - 保证金期末余额 = 期初余额 - 本月保证金留存 - 本月保证金释放。
- **本月新增交易款**：本月内发生的所有交易，即记账时间为当月 1 日至 31 日的所有交易记账金额之和，可与本月交易报表金额核对。
- **本月已结算交易款**：本月已结算的所有交易，即划款时间为当月 1 日至 31 日的所有订单金额。
- **费用**：本月已结算订单所扣取的费用。
  - 内扣费用包含本月所有划款批次，具体可查看每个批次的划款备注。
- **其他**：包含本月所有划款批次的其他款项、返还款项，具体可查看每个批次的划款备注。
- **保证金** = 本月保证金留存 + 本月保证金释放。
  - 本月保证金留存：本月交易款结算留存的保证金。
  - 本月保证金释放：本月已释放划款的保证金金额。
- **本月实划金额**：本月交易款实划金额 + 本月保证金实划金额。报表数据体现为本月已结算交易款 + 费用 + 其他款项 + 保证金。
