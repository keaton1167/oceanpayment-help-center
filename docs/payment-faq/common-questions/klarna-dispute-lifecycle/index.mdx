## 1. Klarna争议处理流程

### 1.1 第一阶段：宽限期

定义：消费者就订单向 Klarna 发起争议后，Klarna 会自动冻结对应账单，并引导消费者按场景处理：

- 退款 / 退货类争议：消费者需完成商品退回。
- 其他类型争议（服务未提供或未收到商品、货不对版/货物损坏、已取消的商品/服务）：消费者需优先主动联系商户协商处理。

处理时限：常规争议场景下，商户拥有 21 天协商周期，可直接与消费者沟通和解，无需 Klarna 介入处理。而针对争议原因为“未授权交易”的订单，商户仅享有 3 天处理时限。

| 争议原因（中文） | 争议原因（英文） | 宽限期时限 |
| --- | --- | --- |
| 退款/退货 | Return | 21天 |
| 服务未提供或未收到商品 | Good not received | 21天 |
| 货不对板/货物损坏 | Faulty Goods | 21天 |
| 已取消的商品/服务 | Incorrect Invoice | 21天 |
| 重复交易 | Already Paid | 21天 |
| 高风险订单 | High risk order | 21天 |
| 受新冠疫情影响 | Pandemic impact | 21天 |
| 未授权交易 | Unauthorized purchases | 3天 |

### 1.2 第二阶段：争议

定义：若宽限期内，商户与消费者未能达成和解，订单将自动进入正式争议流程，由 Klarna 介入核查。平台首先会核验消费者提交的举证材料，判定争议是否成立或予以驳回；如果争议有效，Klarna 会对商户发起提供额外信息回应的请求，要求商户提交相关凭证进行申诉。

争议次数限制和时间限制：每个订单消费者最多可发起 3 次争议，争议原因无额外限制；争议发起时效自交易成功之日起计算，不同争议类型的时效规则如下表所示：

| 争议原因 | 第一次争议（交易成功时间起算） |
| --- | --- |
| 退款/退货 | 180天 |
| 服务未提供或未收到商品 | 180天 |
| 货不对版/货物损坏 | 180天 |
| 已取消的商品/服务 | 180天 |
| 重复交易 | 180天 |

争议请求次数和回复期限：商户需要在期限内回复，逾期未应答将无法再为案件辩护。

| Klarna Request次数 | 商户期限 |
| --- | --- |
| 第1次 | 14天（未授权交易：7天） |
| 后续每次请求 | 7天 |

允许多次发起争议的原因：

- 消费者有新的证据辩护。
- 消费者出于不同的原因提出争议。
- 与对争议案件的关注和重新评估有关。

以下特殊情况，允许超出次数限制发起争议：

- 未授权交易。
- 通过外部机构（如中央仲裁委员会或国家消费者纠纷委员会）提出的法律索赔。
- 当地法律消费者保护规则，如在瑞典，代理人有权再次提出异议。

### 1.3 争议结案判定

商户提交完整举证资料后，Klarna 将综合双方信息作出最终裁定，完成案件结案处理，结果分为两类：

| 争议结果 | 场景 | Klarna状态 | 结案方式 |
| --- | --- | --- | --- |
| 商户争议成功 | 1. 消费者被视为有过错<br />2. 消费者无法提供要求的信息<br />3. 消费者主动取消争议 | Resolved（已解决） | 消费者账单重启 |
| 商户争议失败 | 1. 商户被视为有过错<br />2. 商户逾期未回应争议<br />3. 商户主动接受争议 | Reversal-Merchant Lost（商户败诉） | 商户收到拒付 |
