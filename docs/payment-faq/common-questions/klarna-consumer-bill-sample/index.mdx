以下为 Klarna 消费者端账单页面的主要字段说明，便于商户理解消费者在账单页看到的信息含义：

| 字段 | 说明 |
|---|---|
| Merchant | 网站的账单地址 |
| Purchase date | 支付日期 |
| Order number | Klarna 端的订单号 |
| Billing address | 消费者的账单地址 |
| Delivery address | 消费者的收货地址 |
| Date | 生成账单的日期 |
| Payment reference number | Klarna 端的订单号 |
| Status | 账单状态 |
| Recipient | 银行名称 |
| Payment Method | Klarna 具体的支付方式 |
