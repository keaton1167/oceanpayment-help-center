## 1. 步骤说明

#### 分三步操作：

第一：设置不同的角色，对每个角色分配对应的操作权限菜单

第二：设置不同的数据权限组别，对每个终端号的数据权限进行分组

第三：新增管理员子账户，给不同的子账户配置对应的角色和数据权限

## 2. 具体操作详解

- 登陆Oceanpayment账户后台：
[https://odpm.oceanpayment.com](https://odpm.oceanpayment.com/login)

- 输入在Oceanpayment开立的账户号、登录名、密码
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/EZHkbzfdmosEmZxX514cQxAOn9g.png)

- 成功登录后，点击右上角的设置按钮，则出现设置的菜单，如下图所示：
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/VVhbbwtd6oUqAlxD7z0c1MoInhb.png)

- 点击【操作设置】出现对应菜单，如下图所示：
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/MC31bVNB0oQ5bcxNHJOc4UhQnqh.png)

- 点击【角色】-【添加角色】：可以根据账户使用者的角色或者部门职能，在以下设置页面输入相应的信息。
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/NOyzbiOVLowYkWxP6d3cOisPnEf.png)

- 并在以下位置，勾选需要开通的对应的权限之后，点击“提交”，则角色设置成功。
**注意：** 以下角色功能中“账户设置”权限，如果开通给到子账户，则子账户跟主账户一样拥有管理员最高权限，请慎重开给子账户

![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/ItIOb7GoFo5XSIxfzOOcYyHqnzg.png)

- 【角色】的页面，可以查看当前账户所设置的所有角色列表。并可以针对某个角色的信息或者权限进行修改，或者删除某个角色。
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/XtLkbno36oFANCxzzOWcdlsJnu2.png)

- 【数据权限】-【新增组别】：可以根据不同使用者，分配不同终端号的数据权限。需要开通数据权限的终端号，则进行勾选后点击提交。
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/IlWob6OmFoW86CxygJUcpAt7nHc.png)

- 【数据权限】的页面，可以查看当前所设置的所有数据组别。并可以针对某个数据组别的信息或者对应终端号权限进行修改，或者删除数据组别。
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/OKOYbrCY8oPfOYxRAYWceXzQnNf.png)

- 注意【数据权限】中，按照终端号配置的数据权限，仅适用于以下菜单：
1. 交易查询：此模块下的所有菜单数据
2. 退款管理：此模块下的所有菜单数据
3. 异常交易管理：此模块下的所有菜单数据
4. 网站管理：此模块下的所有菜单数据
5. 除以上菜单外，其他菜单下的数据查询不受限制，可以查询所有终端号的数据

- 【管理员】-【新增管理员】：可以针对员工岗位职能，设置一个子账户的登录名和登录密码。针对子账户的角色权限和数据权限，可以单选也可以多选。
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/Ffgybuq5gojs2nxOFiwcW47znPe.png)

- 【管理员】页面：查看所有增加的操作账户，可以进行修改账户的角色，数据权限，删除账户，重置密码，禁用此账户，账户安全验证重置的操作。
![图片](/img/help-center/oceanpayment-odpm-子账户配置指引-soe3/L7ywbXQFtoRyNKx6yUcc0flCnEb.png)
